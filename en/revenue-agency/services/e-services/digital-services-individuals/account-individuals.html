Help with your account

Common issues for sign-in

Lost password or User ID

Lost password

If you have lost your CRA password, select CRA user ID and then select Forgot your password?

You can create a new password if you correctly answer the security questions you chose when you registered.

Forgotten User ID

To recover your CRA user ID, select CRA user ID and then select Forgot your user ID?

You will need the following:

  • Your SIN
  • Your date of birth
  • The amount you entered on line 15000 of your tax return from one of the 2 previous years

The next time you sign in, you will need to answer 4 of your 5 security questions.

Did not receive one-time passcode

Multi-factor authentication (MFA) increases the security of your account by requiring you to enter a one-time passcode in addition to your password.

One-time passcode by phone

If you cannot receive your one-time passcode by phone, you can:

  • Ask for it to be resent or sent to a different telephone number that is already connected to your account
  • Use the passcode grid if you are already enrolled with that option
  • Confirm that you are signing into your account, make sure auto fill is turned off or direct sign-in is disabled (e.g. through your financial institution)

One-time passcode by passcode grid

If you lose your passcode grid or it expires, you will still be able to sign in if you have also added the telephone option for MFA.

When to contact us

If you still cannot receive your one-time passcode, please contact us by phone.

More help with multi-factor authentication

Expired security code

If your CRA security code expires or has not arrived after waiting 14 days, you can contact us by phone to get a new one.

Recover a locked account

Your account will become locked if you enter the wrong information too many times for any of the following:

  • Password
  • One-time passcode
  • Security questions
  • CRA security code

If your account becomes locked, contact us by phone to have your account unlocked.

Revoked user ID and password

Your user ID and password may be revoked if there is suspicious activity connected with your account.

If this happens, you can:

  • Sign in with another method if you are already registered
  • Register for a new CRA user ID
