About CRA user ID and password

When you choose to access your CRA sign-in account with a CRA user ID and password, you create credentials that you only use with your CRA sign-in accounts. You can use your CRA user ID and password to sign in to My Account, My Business Account, and Represent a Client.

Create a strong CRA user ID and password

When you create your CRA user ID and password, we recommend that you:

  • Make them easy for you to remember and hard for others to guess
  • Avoid using personal information such as your name, social insurance number, mailing or email address
  • Always use unique passwords rather than reusing the same password for different accounts or systems
  • Keep this information secure and do not share it with anyone

The only special characters you can use in your user ID and password are:

  • Dot (.)
  • Dash (-)
  • Underscore (_)
  • Apostrophe (‘)

Creating your CRA User ID

Your CRA user ID must contain between 8 and 16 characters with no spaces, and can contain up to seven numbers.

Each CRA user ID must be unique. You will receive a message if the CRA user ID you create has already been used.

Creating your password

Your password must contain between 8 and 64 characters, including:

  • At least one upper-case letter
  • At least one lower-case letter
  • At least one number
  • No space
  • No accented characters

You cannot use more than 4 consecutive, identical characters.

Manage your CRA user ID and password

Lost or forgotten CRA user ID or password

Forgotten user ID

To recover your CRA user ID, go to CRA sign-in accounts, select CRA user ID from your account sign-in options, and then select Forgot your user ID?

You will need the following:

  • Your Social Insurance Number (SIN)
  • Your date of birth
  • The amount you entered on line 15000 of your tax return from the current or previous tax year

The next time you sign in, you will need to answer 4 of your 5 CRA security questions.

Forgotten password

If you have forgotten your CRA password, go to CRA sign-in accounts, select CRA user ID from your account sign-in options, and then select Forgot your password?

You can create a new password if you correctly answer the security questions you chose when you registered.

Change your CRA user ID or password

You can change your user ID or password after signing in to your account.

  1. Sign in to your account.
  2. Select Profile
  3. Scroll to Security options [Online preferences]
  4. Select Continue

Revoked CRA user ID and password

Your user ID and password may be revoked if there is suspicious activity connected with your account. If that happens, we will contact you. You can regain access to your CRA account in the following ways:

Option 1
Use a different sign in method:

If you have one, you may be able to sign in by using a different CRA user ID and password, Interac® sign-in service, or a provincial partner to access your account.

Option 2
Register for a new CRA user ID:

You may need to Contact us to register for a new CRA user ID and password. When registering, ensure that your user ID is unique to your CRA account. To help avoid having your CRA user ID and password revoked in the future, your new user ID and password should not:

  • Be similar to one that was previously revoked
  • Contain any parts of your name

Once registered, the CRA will mail you a security code to give you full access to online services. While waiting for your security code, you can still file your income tax return online using NETFILE certified software. Alternatively, you may be able to use Interac® document verification service to validate your identity and get full and immediate access to the CRA sign-in services.

How to regain access to your Careers candidate profile

If your CRA user ID and password for the Candidate Profile service have been revoked, you may complete a candidate profile recovery. This will allow the new CRA user ID and password you create during the recovery process to be linked to your existing candidate profile.

If neither option is available to you

You may not be able to use these options if your account is locked or your address on file is incorrect. If so, please contact us by phone.

Choose memorable security questions

You will be asked to choose 5 security questions that will be used verify your identity in different situations, for example, when signing in, if you forget your user ID, or if you sign in from a new device.

Your security questions and answers should be easy for you to remember but hard for others to guess. Answers that can change over time or have more than one answer can be hard to remember.

Change your sign-in option

You must always sign in to your account with the option you registered with.

If you registered with a CRA user ID and password, you can sign in to your account and revoke your credential so you can register with Interac® sign in partner instead.

You can also contact us by phone to revoke your credential so you can register again.

Locked account

If you enter the wrong user ID or password too many times, your account will become locked. If this happens, you will have to contact us by phone to have your account unlocked.
